Основная задача
- отрисовать несколько простых предметов, например, дом, поляна, лес;
- отрисовать солнце и задать ему траекторию перемещения по дуге;
- при заходе солнца за правую границу экрана менять время суток на ночь(затемнить цвета на сцене), затем повторить анимацию, но вместо солнца отрисовывать луну;
- анимация смены времени суток должна быть бесконечной.

Доп. задачи (необязательные)
- по нажатию выбранной кнопки останавливать анимацию;
- по нажатию другой кнопки запускать другую анимацию (например, запускать рост какого-то растению), а по повторному нажатию - останавливать;
- еще какие-то идеи студента.
